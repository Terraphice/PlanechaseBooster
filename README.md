## Planar Atlas
![Planar Atlas](/images/assets/social-preview.jpg)

A complete Planechase experience for Magic: The Gathering — browse a searchable gallery of every plane and phenomenon, build custom decks, and simulate full Planechase games including the Blind Eternities Map mode.

### Features
- **Gallery** — Searchable, filterable card database with grid, singleton, stack, and list views
- **Deck Builder** — Build, name, and save multiple deck slots with import/export
- **Planechase Simulator** — Classic mode and Blind Eternities Map (navigable grid of planes)
- **Custom Cards** — Includes community-created planes alongside official content
- **Profile Seeds** — Export and import your entire setup (preferences + all decks) as a short seed string
