About

A simple gallery and database for Magic: The Gathering's Planechase format.

Advanced search syntax

Use field filters directly in the search box.

  • tag:Plane — match cards with a tag
  • -tag:Phenomenon — exclude a tag
  • name:"Spatial Merging" — search by exact name text
  • /chaos|planeswalker/i — regex search

Keyboard shortcuts

  • / Focus search
  • f Toggle filters
  • g Jump to gallery
  • Tab Accept inline autocomplete
  • n / → Next card in modal
  • p / ← Previous card in modal
  • Esc Close suggestions, modal, settings, or sidebar
