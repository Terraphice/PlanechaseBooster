About

A simple gallery and database for Magic: The Gathering's Planechase format.

Advanced search syntax

Use field filters directly in the search box.

  • name:Akoum — search by card name (alias: n:)
  • o:chaos — search card text (alias: text:, oracle:)
  • tag:Plane — filter by tag (alias: type:, t:)
  • tag:Official — filter by badge label
  • show:hidden — include hidden cards
  • /chaos|annihilator/i — regex search
  • Prefix any filter with - to negate it: -name:Norn
  • Quote multi-word values: name:"Spatial Merging"

Keyboard shortcuts

  • / Focus search
  • f Toggle filters
  • g Jump to gallery
  • Tab Accept inline autocomplete
  • n / → Next card in modal
  • p / ← Previous card in modal
  • Esc Close suggestions, modal, settings, or sidebar
